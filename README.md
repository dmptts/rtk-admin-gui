# Admin GUI

Простой DataGrid позволяющий редактировать, добавлять, удалять и фильтровать записи в таблице.

## Используемые техонологии:

- React
- Typescript
- Redux Toolkit
- React Router
- styled-components
- Formik + yup

## Скрипты

- `npm start` -- Запускает dev-сервер
- `npm run build` -- Собирает проект для продакшена
